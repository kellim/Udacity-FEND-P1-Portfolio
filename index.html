Arcade Game: Bug Crossing

 Bug Crossing is my arcade game for project 3 of Udacity's Front-End Web Development Nanodegree. It is an HTML 5 Canvas powered game created with a game engine and visual assets provided by Udacity. I used object-oriented JavaScript to create and control entities as well as detect collisions.

You can take a look at the repo or play the game.

Website Optimization

 Website Optimization is my fourth project for Udacity's Front-End Web Development Nanodegree.

I improved PageSpeed Insights speed score on a portfolio page provided by Udacity from 28/100 to 94/100 for mobile and from 30/100 to 96/100 for desktop. In addition, I optimized JavaScript code on a pizzeria page with performance issues to run at 60 frames per second. I also automated the build process using Grunt to compress images, minify css and JavaScript files, as well as modify html files prior to copying them to distribution code.

You can take a look at the repo or look at the site that was optimized.. Also, there's the original repo where the code still has all the issues that were fixed.

Neighborhood Map

Farmers Market Finder is my fifth project for Udacity's Front-End Web Developer Nanodegree. For this project, I used Knockout along with the Google Maps API and USDA National Farmers Market Directory API to create a mashup where visitors can search for Farmer's Markets near them by entering their zip code, then nearby Farmer's Markets will be shown on the map.

You can take a look at the repo or use the app.
